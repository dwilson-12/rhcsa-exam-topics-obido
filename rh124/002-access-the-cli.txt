LINUX TO LINUX ACCESS, USE THE SSH COMMAND.

Steps:
1. Get the IP Address first.
	Commands:
		$ ifconfig
		$ ip addr
2. To access the remote machine.
	Commands:
		$ ssh -l [username] [ip address]
